#  Fireplace
Fireplace is a productivity app built for iOS in Swift and Xcode. Stay motivated by collecting woods from every task you complete and keep your fire up every day.

## Features
- Creating tasks
    - Assign custom icons
    - Set the number of woods collected on completion

- Completing tasks
    - Simple on-click completion
    - Accumulate woods collected per day

## Installation

### Notice: 
**this app is built for Xcode to run on iOS devices, and will need to be reinstalled every 7 days for continuous usage without an apple developer account.**
